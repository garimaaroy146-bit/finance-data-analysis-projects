# Finance Data Analysis Projects

This repository contains data analysis projects focused on financial analytics and customer transaction insights.

Projects Included:

## Fraud Detection Analysis
Analyzes transaction patterns to identify fraud indicators and risk factors.

## Credit Risk Analysis
Examines borrower financial characteristics to understand loan default risk.

## Customer Spending Analysis
Explores customer transaction behavior and spending segmentation.

Tools Used:
Python, Pandas, NumPy, Matplotlib, Seaborn
