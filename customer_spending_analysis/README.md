# Customer Transaction Behavior Analysis

## Overview
Banks analyze customer transaction data to understand spending patterns and improve financial services.

This project explores transaction data to identify patterns in customer spending behavior.

## Objectives
- Analyze customer spending patterns across categories
- Examine transaction activity by time of day
- Identify customer spending segments

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
A simulated transaction dataset containing:

- Customer ID
- Transaction Amount
- Spending Category
- Transaction Hour

## Key Analysis
The project analyzes:

- Total spending by category
- Average transaction amount by category
- Transaction activity by hour of the day
- Distribution of transaction amounts
- Customer spending segmentation

## Key Insights
- Spending patterns vary across categories such as food, shopping, travel, and bills.
- Transaction activity tends to increase during evening hours.
- Customer segmentation reveals different spending groups, including low, medium, and high spenders.

## Applications
Customer transaction analysis helps financial institutions:

- Understand customer behavior
- Design personalized financial products
- Improve customer engagement strategies

## Conclusion
The analysis demonstrates how transaction data can provide insights into customer behavior and support data-driven decision-making in financial services.
