# Credit Risk Analysis for Loan Default Prediction

## Overview
Credit risk analysis is an essential process used by banks to evaluate the likelihood that a borrower will default on a loan.

This project analyzes borrower financial attributes to identify patterns related to loan default risk.

## Objectives
- Explore borrower financial data
- Analyze relationships between financial attributes and loan default
- Identify key indicators of credit risk

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
A simulated loan dataset containing:

- Income
- Credit Score
- Loan Amount
- Age
- Default Indicator

## Key Analysis
The analysis investigates:

- Distribution of loan defaults
- Relationship between credit score and default probability
- Relationship between income and loan amount
- Correlation between financial attributes
- Loan-to-income ratio distribution

## Key Insights
- Lower credit scores are associated with higher default probability.
- Borrowers with lower income requesting larger loans may present higher credit risk.
- The loan-to-income ratio is an important metric for assessing borrower risk.

## Applications
Credit risk analytics helps financial institutions:

- Evaluate loan applications
- Manage lending risk
- Improve financial decision-making

## Conclusion
This project demonstrates how financial data analysis can help identify key factors influencing borrower credit risk and support more informed lending decisions.
